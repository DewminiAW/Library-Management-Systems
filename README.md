# 📚 Library Management System
### A Robust Backend Solution for Modern Libraries

This project is a comprehensive **Library Management System** designed to streamline book tracking, member management, and borrowing processes. Developed as part of my Full Stack journey, it showcases clean code principles and efficient database management.

---

## 🛠️ Tech Stack
- **Backend:** Java (JDK 17+)
- **Framework:** Spring Boot / Hibernate ORM
- **Build Tool:** Maven
- **Database:** MySQL
- **Frontend:** HTML/CSS & Bootstrap (Optional/In-progress)

---

## ✨ Key Features
- **Book Management:** Add, update, and delete books with ISBN and Category tracking.
- **Member Portal:** Manage member registrations and borrowing history.
- **Transaction System:** Handle book issues (borrowing) and returns with automated due-date calculations.
- **Search Functionality:** Quickly find books by Title, Author, or Category.
